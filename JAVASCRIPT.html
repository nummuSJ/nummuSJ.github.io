JAVASCRIPT는 웹사이트를 동적으로 만들때에 필수적인 언어이다.

JAVASCRIPT는 웹사이트를 유저친화적으로 만들때에 쓴다.
HTML이 건물의 뼈대와 기본적인 벽,바닥,천장이고
CSS는 건물의 디자인과 페인트 창문들이라면
JAVASCRIPT는 건물에 전기가 통하게끔 회로를 놓고
여러 편의 시설을 구축하는 과정이다.
HTML과 CSS는 웹사이트를 만들기 위해 필요한 언어이지만,
JAVASCRIPT는 만든 웹사이트를 사용자들과 상호작용 할 수 있게끔 만든다는 점에서
프로그래밍 언어라고 부를 수 있다. HTML속성 <scripts> 사이에 자바스크립트 문법으로 작성하거나,
태그 속성 중 이벤트를 담당하는 onclick,onchange 등등의 속성에
작성해서 구현 할 수 있다.

선택자:querySelector('')

자바스크립트에서 태그를 제어하기 위해서는 선택자로 제어 할 태그를 지정해줘야 한다.
선택자로 태그의 속성을 제어할 수 있다.
document.querySelector('') 형식으로 제어할 수 있다.
예를들어 <h1>태그를 제어하고 싶을때는 document.querySelector('h1')으로 지정
이런 경우 모든 <h1> 태그를 제어하게 되는데, 하나의 <h1>만 제어하고 싶을 경우,
class 나 id 를 부여해
document.querySelector('h1.class')
document.querySelector('#지정한아이디')
의 방식으로 제어할 수 있다.

속성을 제어하는 방법은
document.querySelector('h1.class').style.backgroundColor = 'black';
의 방식으로 제어할 수 있다.
주의할 점은 CSS 문법에선 background-color 이지만,
자바 스크립트에서는 backgroundColor 처럼 대문자로 -를 대신한다는 것이다.

조건문:if/else

조건문을 통하여 이벤트를 제어 할 수 있다.
value의 값이 야간모드/주간모드냐에 따라 true/false 값이 반환되어
조건문이 실행되는 모습이다.

if(조건){
조건이 true일때 실행
}
else{
조건이 false일때 실행
}

리팩토링 중복제거

onclick 이벤트를 담고있는 <input>을 선택할때
<input id="target"> 으로 아이디를 지정하고
document.querySelector('#target') 으로 선택할수도 있지만,
this로 종속된 태그를 한번에 선택할수 있다.
또한 다른 태그를 선택하더라도,
var target = document.querySelector('body');
target.style.color = 'black';
처럼 변수를 선언하고 편하게 사용할 수 있다.
